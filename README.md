# 🐉 Interactive Dragon

An interactive dragon animation built using **HTML, CSS, and JavaScript**. This project features a smooth SVG-based dragon that dynamically follows the user's cursor, showcasing creative animations and interactive front-end development techniques.

---

## 🚀 Project Overview

Interactive Dragon is a front-end web project designed to demonstrate:

* 🐉 Interactive Dragon Animation
* 🎨 SVG-Based Visual Design
* ⚡ Smooth User Interactions
* 💻 Front-End Development Skills

The project focuses on creating an engaging and visually appealing dragon animation using core web technologies.

---

## ✨ Features

* 🐉 Interactive Dragon Animation
* 🖱️ Mouse Cursor Tracking
* ⚡ Smooth Real-Time Movements
* 🎨 SVG-Based Graphics
* 🚀 Fast Loading Performance
* 💻 Cross-Browser Compatibility

---

## 🛠️ Technologies Used

| Technology | Purpose                    |
| ---------- | -------------------------- |
| HTML5      | Structure and Content      |
| CSS3       | Styling and Visual Effects |
| JavaScript | Interactive Animations     |
| SVG        | Dragon Graphics Rendering  |
